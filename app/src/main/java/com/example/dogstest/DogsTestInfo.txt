Для взаимодейтсвия клиента с сервером используются JSON - файлы. Они могут хранить строки, примитивные данные, JSON - объекты и массивы.

Для подключения нам надо создать URL, потом открыть соединение через HttpUrlConnection, далее прочитать побайтово запрос, через InputStream, преобразовать байты в символы через
InputStreamReader, далее преобразовать в строку через BufferedReader, далее вернуть строку черезе метод BufferedReader.readLine(). Интернет соединение также нужно делать в
отдельном потоке. В манифесте нужно предупредить пользователя о том, что будет выполняться работа с сетью в данном приложении. Класс StringBuilder позволяет сформировать строку
из других строк. В Java также есть класс JSONObject, из которого мы можем получать какие - либо значения по ключу.
